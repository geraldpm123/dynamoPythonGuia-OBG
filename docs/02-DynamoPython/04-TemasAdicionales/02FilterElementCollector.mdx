# 2. Filtrando Elementos
## 2.1. FilteredElementCollector

Recordemos que Revit es una basede datos, en donde encontraremos muchos objetos, 
de los cuales la mayoria son objetos de la clase Element. Obtener estos elementos peude resultar uan tarea cotidiana.
Para ello Revit API tiene una clase denominada `FilteredElementCollector`,la cual nos sirve para recolectar 
elementos del proyecto de Revit.
## 2.2. Ejemplos
```py
# get category object
obj = System.Enum.ToObject(BuiltInCategory, cat.Id)

# create category filter by category object
filter = ElementCategoryFilter(obj)

# collect filtered elements by category
elements = FilteredElementCollector(doc).WherePasses(filter).WhereElementIsNotElementType().ToElements()
```
